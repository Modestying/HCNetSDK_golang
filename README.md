## 3.使用说明
 编译
 ```
Windows:
	mingw32-make.exe windows
Linux:
	make linux
 ```
 清除
 ```
 Windows:
 	mingw32-make.exe clean
 Linux:
 	make clean
 ```

### 1.Linux遇到错误29
    环境没配好，把lib/linux下的文件扔到/usr/lib/下面就好了
